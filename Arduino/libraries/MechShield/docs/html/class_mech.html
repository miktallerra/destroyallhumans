uint8_t Mech::analogWrite ( int  pin,
int  value 
)

Allows PWM control of the four H bridge outputs on the MechShield. Will generate a steady square wave of the specified duty cycle at a frequency of ~244 Hz.

Note: This can be used to manually control any pin. However, a steady singal on pin 1 is required for proper operation of the servo outputs.
